**TEALS Program** Home | Curriculum Map # TEALS AP CS A Curriculum Assets The TEALS AP CS A curriculum, tests, answer keys, resources and exam can be downloaded from the [Additional Curriculum Materials][] section of the TEALS Dashboard. These items are protected to give the teaching teams the control over when to release these items to their students. Please refrain from sharing these resources publicly in order to ensure the integrity of the curriculum materials. Once you are logged into this resources, navigate to /AP Computer Science A/ folder to see the available resources. ## /Shared resources/ In this folder, you will find resources shared from volunteers and schools across the program over the years. ## /AP Test Prep Materials/ In this document set, you will find resources for students to prepare for the AP exam. ## /College board resources/ In this document set, you will find resources related to the College board. ## /Full Curriculum Package - Releases/ In this folder, you will find the latest full release of the curriculum files including all the latest updates to the curriculum. These releases include project files, solutions, lab documents, starter code, etc.
